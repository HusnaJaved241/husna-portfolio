## How this was built

This site is built entirely in **Flutter**, compiled to a static web app, and
deployed automatically to **GitHub Pages** via **GitHub Actions** ...  meaning
every update I push goes live within a couple of minutes, with zero manual
deployment steps.

**Tech stack:** Flutter · Dart · GitHub Actions · GitHub Pages

It also runs as a native Android/iOS app from the exact same codebase, since
that's one of the advantages of building in Flutter rather than plain HTML.
